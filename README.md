# Engineering Planning

Sprint review dashboards and team planning resources for Konvoy Kegs / Katch Cloud engineering.

## Hosted

This repo is published via GitHub Pages at:
**https://konvoy-kegs.github.io/engineering-planning/**

## Adding a new sprint review

1. Add your HTML dashboard file (e.g. `sprint-review-2026-04-17.html`)
2. Add a card entry in `index.html`
3. Push to `main` — Pages deploys automatically
